### 3. Spacing 5단계 위계 (`docs/spacing-guide.md`)
```
xs(4)   — 같은 그룹 묶기, 인라인 micro
sm(8)   — 항목 사이, 라벨↔값
md(16)  — 그룹↔그룹, 카드 헤더↔본문 (카드 안 모든 분리)
lg(24)  — 카드끼리만
xl(32)  — 다른 섹션·날짜
```
**카드 안에서 lg 쓰지 말 것. 스크롤 압박 = 빈 공간 누적.**

### 4. UI 원칙
- 모든 빈 상태는 `EmptyState` 컴포넌트 (icon + title + subtitle + optional actions)
- 통계 카드는 `SummaryStatRow` 재사용
