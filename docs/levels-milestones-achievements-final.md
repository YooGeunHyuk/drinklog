## 3. 레벨 시스템 (LEVELS · Lv 1~99)

### 3.1 점수 공식 (주도 점수)

```
주도 점수 = 누적ml ÷ 100             [1L = 10pt]
         + 술자리 수 × 100           [1회 = 100pt, 하루 = 한 자리]
         + 연속 음주 보너스           [2일차 +500, … 7일차+ +3000 캡]
         + 고유 카탈로그 수 × 50      [1종 = 50pt]
```

### 3.2 레벨 임계값 곡선

```
thr(n) = round(30 × (n−1)^2.15)
```

| 레벨 | 임계 점수 (근사) |
|---:|---:|
| Lv 10 | ≈ 4,300 |
| Lv 30 | ≈ 40,000 |
| Lv 50 | ≈ 123,000 |
| Lv 70 | ≈ 242,000 |
| Lv 90 | ≈ 384,000 |
| Lv 99 | ≈ 488,000 |
