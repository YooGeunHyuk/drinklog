## 점수 시스템 — 주도(酒道)

```
주도 점수 = 누적 ml + 술자리 수 × 1,000
```

- 누적 ml: 평생 마신 술의 총량
- 술자리 수: 같은 날 여러 잔을 마셔도 1회 (하루 단위 카운트)
- 위스키·와인처럼 양이 적어도 자주 마시면 성장이 이어지도록 설계

**최대치:** 47단계 = 2,167,000점 (≈ 2,170L 수준). 이후 단계는 추후 확장 여지.

## 디자인 의뢰 시 주의사항

### 톤 일관성

### 컬러 시스템
- 각 뱃지는 `BG` + `Text` 색 페어를 가짐
- 일러스트는 이 색 안에서 표현 가능해야 함 (배경에 깔리는 색이 되어야 함)
- 후반 아크(8–9)는 어두운 BG + 금색/보라 Text — 신화적 느낌

### 표현 톤
- 캐리커처/유머 — 과도한 조롱은 NO, 자조적 공감 OK
- Arc 4–5 (무너진다/이불킥) — 너무 어둡거나 자극적이면 안 됨, 약간 코믹하게
- Arc 8–9 (달관/전설) — 동양화 느낌, 묵화·서예 톤 가능

### 사이즈
- 메인 사이즈 512×512 (앱 내 표시는 작게)
- 라운드 처리된 정사각형 또는 원형 뱃지 형태 권장
- 투명 배경 PNG 또는 색 BG 포함 PNG 둘 다 필요할 수 있음
